# Zombie

使用Gobot框架制作的树莓派工具,能够展示来自PC的传感器数据,简易的树莓派信息以及房间温湿度并显示到一个微型OLED上.

An tool made by Gobot framework. Can read some sensor data and display on a SSD1306 OLED.

> Raspberry Pi  
> SSD1306  
> SHT30  
> Open Hardware Monitor  
> Gobot  
> fogleman/gg  
> hajimehoshi/bitmapfont
> shirou/gopsutil  
> robfig/cron  
> go-redis/redis  
> bitly/go-simplejson
